使用strace查看系统调用情况

发表于 2015-09-15   |  

strace是linux下一个很有用的工具，可以用来跟踪某个命令/进程的系统调用以及接受的信号情况。

strace的典型使用如下

  • 列出某个命令所有的系统调用
  • 列出某个命令特定的系统调用
  • attach到当前某个进程下，打印每一个系统调用
  • 打印系统调用的时间戳
  • 打印系统调用的执行时长
  • 汇总统计信息

执行过程中的系统调用

这是strace最基础的功能。直接strace command即可

只列出特定的系统调用

这个可以strace command之后，grep一下，也可以使用-e参数指定

使用strace -e trace=open,read command 指定多个系统调用

attach到某个当前进程下

-p参数可以指定进程号，-o 让结果输出到文件里

打印系统调用的时间戳

-t可以打印系统调用的时间戳

打印系统调用的执行时长

-r可以打印各个系统调用的执行时长

统计各系统调用汇总信息

-c可以得到一些汇总的统计信息

可以看到各种调用的时间占比，具体占用时间，调用次数，错误次数等等信息。
